# Component Patterns

## Form Fields
- Labels always above the input (never inline/floating)
- Label: 13px, 500 weight, 6px bottom margin
- Error messages appear below the field in red (#ef4444), 12px
- Required fields are not marked -- optional fields get "(optional)" suffix

## Buttons
- **Primary**: White text on Indigo 500, full-width in forms, 15px/600
- **Secondary**: Gray 900 text on white, 1px Gray 200 border
- **Ghost**: Gray 500 text, no background, underline on hover
- Minimum touch target: 44px height

## Cards
- White background, 1px Gray 200 border, 12px radius
- 24px internal padding
- Card shadow: 0 1px 3px rgba(0, 0, 0, 0.04)
- Cards never nest inside other cards

## Spacing Scale
- 4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 48px
- Default gap between form fields: 16px
- Default gap between sections: 24px
