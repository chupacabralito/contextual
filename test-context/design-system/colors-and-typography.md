# Colors & Typography

## Primary Colors
- **Indigo 500**: #6366f1 -- Primary actions, links, focus states
- **Indigo 600**: #4f46e5 -- Hover state for primary actions
- **Indigo 50**: #eef2ff -- Light backgrounds for selected states

## Neutral Palette
- **Gray 900**: #1d1d1f -- Primary text
- **Gray 500**: #6e6e73 -- Secondary text, placeholders
- **Gray 200**: #e5e5ea -- Borders, dividers
- **Gray 50**: #f5f5f7 -- Page background

## Typography
- **Font stack**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **Heading 1**: 28px / 700 weight / #1d1d1f
- **Heading 2**: 20px / 600 weight / #1d1d1f
- **Body**: 14px / 400 weight / #1d1d1f / 1.6 line-height
- **Caption**: 12px / 400 weight / #86868b

## Component Tokens
- **Border radius**: 8px (inputs), 12px (cards), 4px (badges)
- **Shadow (card)**: 0 1px 3px rgba(0, 0, 0, 0.04)
- **Input padding**: 10px 12px
- **Button padding**: 12px 20px
